About This Application

This is a complete Flask application that demonstrates all the concepts learned throughout the course.

Technologies Used

  • Flask - Web framework
  • SQLAlchemy - Database ORM
  • Flask-WTF - Forms and CSRF protection
  • Werkzeug - Password hashing
  • pytest - Testing framework
  • Gunicorn - Production server

Features Implemented

  • Application Factory Pattern
  • Blueprints for code organization
  • User authentication with secure passwords
  • Blog post management (CRUD operations)
  • RESTful API endpoints
  • Security features (CSRF, XSS prevention)
  • Comprehensive testing
  • Production-ready configuration

Project Structure

app/
├── __init__.py      # Application factory
├── models.py        # Database models
├── config.py        # Configuration
├── auth/            # Authentication blueprint
├── main/            # Main pages blueprint
├── posts/           # Posts blueprint
└── api/             # API blueprint
