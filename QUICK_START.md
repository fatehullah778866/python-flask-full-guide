## Step 2: Set Up Your Environment

### Install Python
Make sure you have Python 3.8+ installed:
```bash
python --version
```

### Create a Virtual Environment
```bash
# Create virtual environment
python -m venv venv

# Activate it
# On Windows:
venv\Scripts\activate
# On Mac/Linux:
source venv/bin/activate
```

### Install Flask
```bash
pip install flask
```

## Step 3: Start Learning

### Begin with Module 1
1. Open [LEARNING_PLAN.md](./LEARNING_PLAN.md)
2. Find Module 1: Flask Fundamentals
3. Work through each topic in order
4. Build the Module 1 project

### How to Get Help
When you're ready to learn a specific topic:

## Step 4: Practice Regularly

- Code every day (even 30 minutes helps!)
- Build all the projects
- Experiment and try variations
- Don't skip ahead - master fundamentals first

## Recommended Learning Flow

1. **Read** the topic in LEARNING_PLAN.md
2. **Ask me** to explain or demonstrate the concept
3. **Code** along with examples
4. **Build** the module project
5. **Review** and understand what you built
6. **Move** to the next topic
