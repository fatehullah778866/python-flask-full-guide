Available Endpoints

GET /api/items

Get all items

curl http://127.0.0.1:5000/api/items

GET /api/items/<id>

Get single item by ID

curl http://127.0.0.1:5000/api/items/1

POST /api/items

Create new item

curl -X POST http://127.0.0.1:5000/api/items -H "Content-Type: application/json" -d '{"name": "New Item", "description": "Description"}'

PUT /api/items/<id>

Update item by ID

curl -X PUT http://127.0.0.1:5000/api/items/1 -H "Content-Type: application/json" -d '{"name": "Updated Item"}'

DELETE /api/items/<id>

Delete item by ID

curl -X DELETE http://127.0.0.1:5000/api/items/1

How to Test

  1. Use curl commands in terminal
  2. Use Postman or similar API tools
  3. Use browser for GET requests
  4. Check the response JSON
