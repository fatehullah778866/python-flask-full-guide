How It Works

  1. Submit Task: Click "Submit Task" to start a background job
  2. Get Task ID: You'll receive a task ID immediately
  3. Check Status: View task progress on the status page
  4. Background Processing: Task runs in separate worker process
  5. No Blocking: App remains responsive while task runs!

Setup Instructions

To run this app, you need:

  1. Install Redis: brew install redis (Mac) or download from redis.io
  2. Start Redis: redis-server
  3. Start Celery Worker: celery -A app.celery worker --loglevel=info
  4. Start Flask App: python app.py
